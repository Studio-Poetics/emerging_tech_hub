What is a blockchain and how does it work?

Click to reveal answer
01 Foundation
Core Concept
A blockchain is a distributed, immutable ledger that records transactions across multiple computers in a way that makes the records extremely difficult to alter, hack, or cheat.
Key Principle
Each block contains cryptographic hashes linking to the previous block, creating an unbreakable chain. Consensus mechanisms ensure all network participants agree on the ledger's state.
Example
Bitcoin blockchain: Each transaction is grouped into blocks, verified by miners, and permanently recorded, creating a transparent and tamper-proof payment system.
