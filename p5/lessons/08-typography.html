Lesson 8

Typography

Text is not just communication - it's form. Letters have shape, weight, rhythm, and texture. In computational design, typography becomes dynamic, responsive, programmable.

Drawing Text

The text() function draws text on the canvas:

text('HELLO', 200, 200);

The first parameter is the text to display (in quotes). The next two are x and y position.

Key Concept: Text in code is called a "string" and must be wrapped in quotes: 'like this' or "like this"

Text Properties

Size

textSize(32); // Set size in pixels text('BIG', 200, 200);

Alignment

Control how text is positioned relative to its x,y coordinates:

textAlign(CENTER, CENTER); // Horizontal, Vertical text('CENTERED', 200, 200);

Options: LEFT, RIGHT, CENTER for horizontal. TOP, BOTTOM, CENTER, BASELINE for vertical.

Fill and Stroke

Like shapes, text has fill and stroke:

fill(0); // Black fill stroke(100); // Gray outline strokeWeight(2); text('OUTLINED', 200, 200);

Dynamic Text

You can use variables in text, combining strings with values using the + operator:

let count = 42; text('Count: ' + count, 200, 200);

This lets you display changing information like frame count, mouse position, or any calculated value.

Text as Visual Element

Think of text not just as words, but as graphic form:

  • Repeat letters in patterns using loops
  • Make text size respond to interaction
  • Position letters individually as compositional elements
  • Use single characters as abstract shapes

Experiment

  • Change the text and size
  • Try different alignments: textAlign(LEFT), textAlign(RIGHT)
  • Make text follow the mouse
  • Create a grid of repeating letters using a loop
  • Make the size change based on mouseX using map()

Challenge: Can you create a composition using just the letter 'O' at different sizes? Or make text that grows as the mouse approaches it?

Typographic Thinking

Designers like Muriel Cooper and John Maeda explored dynamic typography - type that moves, responds, and transforms. You now have the tools to continue this exploration. Typography is not static - it can be fluid, reactive, generative.
