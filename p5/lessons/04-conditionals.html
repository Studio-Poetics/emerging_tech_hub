Conditional Statements

Code can make decisions. Based on conditions, your program can choose different paths, creating adaptive, responsive behavior.

The if Statement

The if statement lets you run code only when a condition is true:

if (mouseX < 200) { // This code runs only when mouseX is less than 200 circle(100, 200, 50); }

Key Concept: The condition goes in parentheses ( ). The code to run goes in curly braces { }. If the condition is true, the code runs. If false, it's skipped.

Comparison Operators

Use these to compare values:

  • < less than
  • > greater than
  • <= less than or equal to
  • >= greater than or equal to
  • === equal to (three equals signs!)
  • !== not equal to

if-else: Choosing Between Two Options

Use else to specify what happens when the condition is false:

if (mouseX < 200) { // Left side behavior rect(150, 150, 100, 100); } else { // Right side behavior circle(200, 200, 100); }

Logical Operators

Combine multiple conditions:

  • && AND - both conditions must be true
  • || OR - at least one condition must be true
  • ! NOT - inverts true/false
// Only true if BOTH are true if (mouseX > 100 && mouseX < 300) { // Mouse is in the middle range }

else if: Multiple Choices

Check multiple conditions in sequence:

if (mouseX < 133) { // Left third } else if (mouseX < 266) { // Middle third } else { // Right third }

Experiment

Move your mouse across the canvas and watch the shape change. Try:

  • Change the boundary from 200 to a different value
  • Add more conditions with else if
  • Use mouseY instead of mouseX
  • Combine conditions with && to divide the canvas into quadrants

Challenge: Can you create a composition that changes completely based on where the mouse is? Different shapes, different positions - create four distinct visual states for four quadrants of the canvas.

Conditional Thinking

Conditionals are about creating rules and logic. You're not just drawing - you're defining when and how things happen. This is the foundation of interactive systems and artificial intelligence.
