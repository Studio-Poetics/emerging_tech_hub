Shapes

Every complex visual composition begins with simple forms. Circles, rectangles, lines, and triangles are your fundamental vocabulary for creating meaning and beauty.

Basic Shape Functions

P5.js provides simple functions for drawing basic geometric shapes. Each function requires specific parameters that define the shape's position and size.

Circle

circle(x, y, diameter);

Draws a circle at position (x, y) with the specified diameter.

Example: circle(200, 150, 80); draws a circle at x=200, y=150 with diameter 80.

Rectangle

rect(x, y, width, height);

Draws a rectangle starting at position (x, y) with the specified width and height.

Line

line(x1, y1, x2, y2);

Draws a line from point (x1, y1) to point (x2, y2).

Triangle

triangle(x1, y1, x2, y2, x3, y3);

Draws a triangle connecting three points.

Key Concept: Think of these shapes not as isolated objects, but as elements you can compose together. The Bauhaus and De Stijl movements explored how simple geometric forms could create complex visual meaning.

Controlling Appearance

Fill Color

fill() sets the interior color of shapes. Like background(), you can use a single number for grayscale:

fill(0); // Black fill fill(128); // Gray fill fill(255); // White fill

Stroke (Outline)

stroke() sets the color of the outline. strokeWeight() sets the thickness in pixels.

stroke(0); // Black outline strokeWeight(3); // 3 pixels thick

Removing Fill or Stroke

noFill(); - Draw only outlines (no interior color)
noStroke(); - Draw filled shapes without outlines

Experiment

Try modifying the code:

  • Move the shapes by changing their x and y coordinates
  • Change their sizes (diameter, width, height)
  • Add noStroke(); before drawing shapes
  • Change fill colors using different numbers (0-255)
  • Draw additional shapes - try line() or triangle()

Challenge: Can you create a simple face using only circles and lines? Or a geometric composition inspired by Piet Mondrian?
