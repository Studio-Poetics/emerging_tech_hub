Lesson 5

Loops

Repetition is fundamental to pattern. Instead of drawing each element manually, loops let you define a rule and repeat it - creating complex compositions from simple instructions.

The for Loop

A for loop repeats code a specific number of times. It's perfect for creating patterns and grids.

for (let x = 50; x <= 350; x += 50) { circle(x, 200, 40); }

This draws circles at x = 50, 100, 150, 200, 250, 300, 350.

Key Concept: A for loop has three parts:
1. let x = 50 - start value
2. x <= 350 - continue while this is true
3. x += 50 - increase x by 50 each time

Understanding the Loop Variable

The variable (often called i, x, or y) changes each time through the loop. You can use it to control position, size, or any property:

// Position changes for (let x = 0; x < 400; x += 40) { circle(x, 200, 30); } // Size changes for (let i = 0; i < 5; i++) { let size = (i + 1) * 20; circle(200, 200, size); }

Nested Loops: Creating Grids

Put one loop inside another to create two-dimensional patterns:

for (let x = 50; x < 400; x += 50) { for (let y = 50; y < 400; y += 50) { circle(x, y, 30); } }

The outer loop controls columns (x position). The inner loop controls rows (y position). Together they create a grid.

Common Loop Patterns

Count from 0 to N:

for (let i = 0; i < 10; i++) { // Runs 10 times: i = 0, 1, 2, ... 9 }

Step by specific amount:

for (let x = 0; x < 400; x += 20) { // x = 0, 20, 40, 60, ... }

Count backwards:

for (let i = 10; i > 0; i--) { // i = 10, 9, 8, ... 1 }

Experiment

  • Change the spacing: try x += 30 or x += 80
  • Change the start and end values
  • Make the circles grow in size as x increases
  • Create a nested loop to make a grid
  • Try making a diagonal line by linking x and y positions

Challenge: Can you create a grid where circle size is based on distance from the center? Or a pattern where each row has one more circle than the previous row?

Repetition as Composition

Artists like Sol LeWitt used simple rules of repetition to create complex visual work. Loops are your tool for this kind of systematic, rule-based art. You define the logic, the computer executes the pattern.
