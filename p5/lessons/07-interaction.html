Lesson 7

Interaction

Your work can listen and respond. Interaction creates dialogue between the person and the system - the computer becomes not just a display, but a conversational partner.

Mouse Position Variables

P5.js continuously tracks the mouse position and makes it available through two special variables:

  • mouseX - The current x-coordinate of the mouse
  • mouseY - The current y-coordinate of the mouse

You can use these just like any other number:

circle(mouseX, mouseY, 50);

This draws a circle that follows your mouse cursor.

Key Concept: These variables update automatically every frame. You don't need to do anything special - just use them in your code.

Detecting Mouse Clicks

mouseIsPressed

This variable is true when the mouse button is pressed, false when it's not. You can use it with an if statement:

if (mouseIsPressed) { // Code here runs only when mouse is pressed circle(mouseX, mouseY, 20); }

mousePressed() Function

This special function runs once each time the mouse is clicked:

function mousePressed() { // Code here runs once per click background(0); }

Mapping Values

The map() function is incredibly useful for interaction. It converts a value from one range to another:

let size = map(mouseX, 0, 400, 20, 200);

This takes mouseX (which ranges from 0-400) and converts it to a size (which will range from 20-200). As the mouse moves right, the size increases.

Other Useful Variables

  • pmouseX, pmouseY - Previous mouse position (from last frame)
  • key - The last key pressed on the keyboard
  • keyIsPressed - True when any key is pressed

Experiment

Move your mouse over the canvas on the right and see what happens. Then try:

  • Change the circle size from 50 to something else
  • Make the circle change size based on mouseX using map()
  • Draw at the mouse position only when the mouse is pressed
  • Use both mouseX and mouseY to control different properties

Challenge: Can you create a simple drawing tool where clicking creates circles at the mouse position? Or make the background color change based on mouse position?

Building Responsive Systems

Interaction is not just about following the mouse. It's about creating systems that respond meaningfully to input - systems that feel alive and conversational. The best interactive work creates unexpected relationships between input and output.
