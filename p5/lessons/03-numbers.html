Lesson 3

Drawing with Numbers

Numbers are not just values - they are relationships. When you store values in variables and use math to create relationships between them, you begin thinking computationally.

Variables: Storing Values

Instead of using numbers directly, you can store them in variables. This makes your code more flexible and easier to understand.

let circleSize = 80; circle(200, 200, circleSize);

Now you can use circleSize anywhere in your code, and changing it in one place updates it everywhere.

Key Concept: Use let to create a variable. Give it a meaningful name that describes what it stores. Names cannot have spaces - use camelCase like circleSize instead of circle size.

Built-in Variables

P5.js provides useful variables that update automatically:

  • width - The width of your canvas
  • height - The height of your canvas

These are perfect for creating responsive layouts:

let centerX = width / 2; // Horizontal center let centerY = height / 2; // Vertical center

Math Operations

You can use math to calculate values:

  • + addition: 100 + 50 = 150
  • - subtraction: 200 - 75 = 125
  • * multiplication: 20 * 3 = 60
  • / division: 400 / 2 = 200

Creating Relationships

The power of variables and math is creating relationships. Instead of hardcoding every value, you can define them in terms of each other:

let baseSize = 50; let smallSize = baseSize * 0.5; // Half the base let largeSize = baseSize * 2; // Twice the base

Now if you change baseSize, all the related sizes update proportionally. This is computational thinking - defining systems of relationships, not just individual values.

Experiment

  • Change circleSize and see everything update
  • Try using width and height to center shapes
  • Create a variable for spacing and use it to position multiple shapes
  • Can you make three circles where each is 1.5x larger than the previous?

Insight: You are not just typing numbers. You are designing relationships and systems. This is the beginning of algorithmic thinking.
