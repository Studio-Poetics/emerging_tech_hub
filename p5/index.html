Code as Medium

Learning to Code with P5.js

The computer is a medium for creative expression. Just as painters use brushes and canvas, sculptors use clay and tools, you can use code to create, explore, and express ideas.

This is an introduction to programming through P5.js, designed specifically for designers and artists who are encountering code for the first time. You will learn the fundamentals of computational thinking by making visual, interactive work.

"The computer is a drawing tool. We draw with it, and sometimes it draws with us."

Course Structure

This course contains 9 comprehensive lessons that progress from basic concepts to creative systems. Each lesson includes interactive code examples that you can modify in real-time to see immediate results.

Lesson 1

The Canvas

Understanding digital space. The canvas is your drawing surface, and every point has a precise location in the coordinate system.

Lesson 2

Shapes

Building blocks of visual composition. Circles, rectangles, lines, and triangles are your visual vocabulary.

Lesson 3

Drawing with Numbers

Variables and mathematical thinking. Learn to create relationships between values and think computationally about space and form.

Lesson 4

Loops

Repetition and pattern. Use loops to create grids, sequences, and systematic compositions from simple rules.

Lesson 5

Conditional Statements

Making decisions in code. Create work that adapts and responds based on conditions and logic.

Lesson 6

Motion

Time and change. Computation brings movement to your work through the animation loop, creating living, breathing compositions.

Lesson 7

Interaction

Responsive systems. Your work can listen and respond to mouse movement, clicks, and keyboard input, creating dialogue between user and machine.

Lesson 8

Typography

Text as visual element. Explore dynamic, responsive typography and treat letters as programmable graphic forms.

Lesson 9

Variation

Controlled randomness and generative thinking. Create systems that produce infinite variations within defined constraints.

About This Course

This curriculum follows the philosophy of computational art pioneers like John Maeda and Casey Reas, who view programming not as engineering but as a creative medium with its own unique properties and possibilities.

P5.js is a JavaScript library that makes coding accessible and visual. Created by Lauren McCarthy, it builds on the work of Processing (created by Casey Reas and Ben Fry) and brings creative coding to the web browser.

How to Use This Course

  • Work through lessons in order - each builds on previous concepts
  • Experiment freely with the code examples - you cannot break anything
  • Type slowly and observe what happens as you change values
  • Use the Reset button to restore original code if needed
  • There are no wrong answers - exploration is the goal
